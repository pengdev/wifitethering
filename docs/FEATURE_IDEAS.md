# Feature Brainstorming & Revenue Roadmap

This document outlines potential features to enhance the "Tethering Shortcut" app's value proposition and drive revenue through Ads and Premium IAP.

## 💰 Monetization Strategy

| Tier | Access | Key Features |
| :--- | :--- | :--- |
| **Free** | Ad-supported | Basic Toggle, Connected Devices (Live), Speed Test, QR Sharing. |
| **Premium** | One-time IAP | Ad Removal, Smart Limits, Advanced Automation, Premium Themes, Widgets. |

---

## 🛠 Feature Roadmap

### 1. Smart Limits (Premium Bundle)
*Targeting: Battery preservation and data overage prevention.*
- **Data Cap:** Automatically turn off hotspot (or notify) once a specific data limit (e.g., 500MB, 1GB) is reached.
- **Battery Protector:** Automatically shut down the hotspot if the phone's battery drops below a user-defined threshold (e.g., 15%).
- **Idle Timeout:** Detect when no devices are connected for X minutes and automatically turn off the hotspot to save power.

### 2. Advanced Automation (Premium)
*Targeting: Power users and commuters.*
- **Bluetooth Trigger:** Automatically enable the hotspot when the phone connects to a specific Bluetooth device (e.g., Car BT, Tablet BT).
- **Charger Trigger:** Option to only allow hotspot while the device is charging.
- **Location-based Reminders:** Send a notification to enable hotspot when arriving at a specific location (Work/Home).

### 3. UI/UX & Quality of Interest
- **Quick Settings Tile (Free):** A dedicated Android system tile to toggle the hotspot directly from the notification shade.
- **Home Screen Widgets (Premium):** Customizable widgets showing current status, connected device count, and a quick toggle button.
- **Premium Themes (Premium):** Unlock aesthetic "Glassmorphism" or "Cyberpunk" themes beyond the standard Material 3 design.
- **App Shortcuts (Free):** Long-press the app icon to toggle the hotspot instantly.

### 4. Network Tools & Sharing
- **QR Code Sharing (Free):** Generate a QR code within the app so friends/family can scan and connect instantly without typing passwords.
- **Speed Test (Ad-supported):** A built-in tool to measure the upload/download speed of the tethered connection.
- **Device Naming (Premium):** Allow users to nickname recognized MAC addresses (e.g., "John's iPad") for easier management.

## 📈 Next Steps for Implementation

1.  **Phase 6.1 (Quick Wins):** Implement the **Quick Settings Tile** and **QR Code Sharing** to increase user engagement.
2.  **Phase 6.2 (Revenue Focus):** Build the **Battery/Data Limiters** and gate them behind the Premium IAP.
3.  **Phase 6.3 (Polish):** Add **Home Screen Widgets** and **Premium Themes** to make the IAP more visually rewarding.
